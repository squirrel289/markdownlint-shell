# SH001/shell-tree-sync

## Description

`SH001` ensures fenced shell tree blocks stay synchronized with the current filesystem tree.

## Failing Example

```markdown
```bash tree --noreport -F --charset utf-8 -L 2 demo
stale-output
```
```

## Passing Example

```markdown
```bash tree --noreport -F --charset utf-8 -L 2 demo
demo/
`-- sample.txt
```
```

## Configuration

- `include_backlog` (`boolean`, default `false`): include `backlog/` markdown paths.
- `annotations_file` (`string`, optional): explicit path to annotation YAML.

## Notes

When markdownlint runs with `--fix`, this rule rewrites stale tree block bodies to current output.
