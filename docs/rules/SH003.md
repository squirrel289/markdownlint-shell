# SH003/shell-tree-unused-annotation

## Description

`SH003` reports annotation labels that do not map to any rendered tree entry.

## Failing Example

```yaml
{demo}:
  missing.txt: Unused note
```

for:

```markdown
```bash tree --noreport -F --charset utf-8 -L 2 demo {demo}
...
```
```

## Passing Example

```yaml
{demo}:
  sample.txt: Example note
```

where `sample.txt` exists in the rendered tree block.

## Configuration

- `annotations_file` (`string`, optional): explicit path to annotation YAML.
