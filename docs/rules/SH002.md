# SH002/shell-tree-annotation-selector

## Description

`SH002` validates annotation selectors (for example `{demo}`) used in shell tree blocks.

## Failing Example

```markdown
```bash tree --noreport -F --charset utf-8 -L 2 demo {missing}
...
```
```

The selector fails when no matching section exists in `.bash-tree.yaml`/`.bash-tree.yml`.

## Passing Example

```markdown
```bash tree --noreport -F --charset utf-8 -L 2 demo {demo}
...
```
```

with:

```yaml
{demo}:
  file.txt: File note
```

## Configuration

- `annotations_file` (`string`, optional): explicit path to annotation YAML.
